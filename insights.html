01 DESIGN_LOGIC

The Bezel Rule

Visual symmetry is the hallmark of professional hardware. In the Jungle, we replicate the ThinkPad's physical balance. Every element on the left must have a counter-weight on the right to maintain cognitive focus.

02 UX_UI

The Copy-Paste Trap

Convenience shouldn't break core function. If a script captures every click to copy text, it effectively kills the ability to travel to new pages. True power lies in selective automation.

03 COLOR_THEORY

GitHub Blue

High-saturation colors cause chromostereopsis (eye strain). Using #58a6ff signals a professional environment. It is the "native" color of the developer, built for long night-shift sessions.

04 ARCHITECTURE

The Narrative Flow

Information architecture is storytelling. Order topics from "Student" to "Business Owner" to create a psychological sense of progress. Logic must always precede the sale.

05 PSYCHOLOGY

The Cursor Silence

A blinking cursor is an active request. Removing it once the typing is finished gives the user "permission" to stop reading and start acting. It is the visual equivalent of a period.

06 PHILOSOPHY

The Unix Pivot

Technical choices define the project's soul. Choosing "Unix" over "Linux" respects the ancestry of OpenBSD. It signals a "Security by Default" mindset that the masses often overlook.

07 GEOMETRY

The Grid Void

Symmetry is a visual promise to the user. An incomplete grid feels broken. Always finish the row (the Power of 20) to maintain the "Command Center" authority.

08 NAVIGATION

Intent Mapping

Distinguish between "Snippet Links" (for tools) and "Destination Links" (for knowledge). Never force a user to choose between copying and navigating. The path must be clear.
